### Express应用生成器

通过应用生成器express可以快速创建一个应用的骨架.

###### 安装

因为不需要存在于项目中,只是个工具,所以安装在全局就行.

```powershell
npm i express-generator -g
```

创建脚手架项目

```powershell
express scaffold-demo --view=ejs
```

创建的脚手架项目默认是不会给你下载依赖包的,所以我们还需要进入项目里进行依赖安装.

```powershell
cd scaffold-demo
npm i
```

启动并运行项目

```powershell
npm start
```
