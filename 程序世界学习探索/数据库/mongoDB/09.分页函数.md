### 分页方法

cursor.limit()	取多少个文档

cursor.skip()	跳过多少个文档

需求:取出前5个文档

```
 cursor.limit(5)
```

需求:要求跳过前面的文档,取出剩余的所有文档

```
cursor.skip(5)
```



注意点:我们可以直接在find方法后面调用limit方法或者skip方法

```
 db.person.find().limit(5)
```

MongoDB是支持链式调用的

```
 db.person.find().skip(5).limit(5)
```



注意点:skip函数永远在limit函数之前执行

```
db.person.find().limit(5).skip(5) //效果和上面一样
```
